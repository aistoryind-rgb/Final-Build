# LifeInvader EN3 Advertisement Editor - V2 (Modular & Clean Refactored)

An internal advertisement processing and validation tool for GTA 5 Grand RP (EN3).

## Key Features
- **Data-Driven Architecture**: All item, vehicle, and clothing databases are decoupled from codebase into standard JSON format.
- **Performant & Offline-First**: Instant validation without API dependencies unless utilizing the optional Gemini Spark assistant.
- **Modular Codebase**: Split into structured logic files (`db.js`, `engine.js`, `ui.js`).
